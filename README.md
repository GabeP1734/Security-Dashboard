# Security Dashboard

A full stack security dashboard built with Flask, SQLite, HTML, CSS, and JavaScript. The application simulates core security operations workflows such as alert management, log parsing, and network scanning.

## Features

- Session based user authentication
- Manual alert creation and deletion
- Log file upload and parsing for suspicious activity
- Severity based alert filtering
- Basic port scanning for common services
- Scan history tracking
- Input validation for scanner input

## Tech Stack

- Python
- Flask
- SQLite
- HTML
- CSS
- JavaScript

## Project Purpose

This project was built to demonstrate full stack development skills in a cybersecurity context. It combines web development, backend logic, persistent storage, and security themed functionality into one application.
